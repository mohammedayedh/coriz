# خطة إضافة مصادر OSINT بدون API

## الهدف
إضافة مصادر معلومات قوية تعمل بدون API من خلال Web Scraping، مع التركيز على المصادر التي تعطي معلومات حقيقية وكثيرة.

## المصادر المختارة (بدون API - مجانية 100%)

### المرحلة الأولى: مصادر الشبكات الاجتماعية

#### 1. Instagram OSINT (instagram-osint)
- **الوصف**: جمع معلومات من Instagram بدون API
- **البيانات**: الملف الشخصي، المتابعين، المنشورات، الهاشتاغات
- **الطريقة**: Web scraping من الصفحات العامة
- **الأداة**: Instalooter / Instagram-scraper
- **المعلومات**: صور، فيديوهات، تعليقات، إعجابات، موقع جغرافي

#### 2. TikTok OSINT (tiktok-osint)
- **الوصف**: جمع معلومات من TikTok
- **البيانات**: الفيديوهات، التعليقات، الإحصائيات
- **الطريقة**: Web scraping
- **الأداة**: TikTok-scraper
- **المعلومات**: محتوى الفيديو، الموسيقى، الهاشتاغات

#### 3. Reddit OSINT (reddit-osint)
- **الوصف**: جمع معلومات من Reddit
- **البيانات**: المنشورات، التعليقات، المجتمعات
- **الطريقة**: Web scraping من الصفحات العامة
- **المعلومات**: تاريخ النشاط، الاهتمامات، الآراء

#### 4. YouTube OSINT (youtube-osint)
- **الوصف**: جمع معلومات من YouTube
- **البيانات**: القنوات، الفيديوهات، التعليقات
- **الطريقة**: Web scraping + youtube-dl
- **المعلومات**: محتوى القناة، الإحصائيات، المشتركين

### المرحلة الثانية: مصادر البحث والأرشفة

#### 5. Google Dorks (google-dorks)
- **الوصف**: بحث متقدم في Google
- **البيانات**: ملفات مكشوفة، معلومات حساسة
- **الطريقة**: استعلامات بحث متقدمة
- **المعلومات**: مستندات، قواعد بيانات، ملفات تكوين

#### 6. Wayback Machine (wayback-machine)
- **الوصف**: أرشيف المواقع القديمة
- **البيانات**: نسخ قديمة من المواقع
- **الطريقة**: Archive.org API (مجاني)
- **المعلومات**: تاريخ الموقع، التغييرات، المحتوى المحذوف

#### 7. Pastebin Search (pastebin-search)
- **الوصف**: البحث في التسريبات
- **البيانات**: كلمات مرور، بيانات مسربة
- **الطريقة**: Web scraping من Pastebin وبدائله
- **المعلومات**: تسريبات، أكواد، معلومات حساسة

### المرحلة الثالثة: مصادر النطاقات والشبكات

#### 8. DNS History (dns-history)
- **الوصف**: تاريخ سجلات DNS
- **البيانات**: سجلات DNS القديمة
- **الطريقة**: SecurityTrails (free tier) + Web scraping
- **المعلومات**: تغييرات DNS، النطاقات الفرعية القديمة

#### 9. Certificate Transparency (cert-transparency)
- **الوصف**: شهادات SSL/TLS
- **البيانات**: النطاقات الفرعية من الشهادات
- **الطريقة**: crt.sh (مجاني)
- **المعلومات**: جميع النطاقات الفرعية، تواريخ الإصدار

#### 10. WHOIS History (whois-history)
- **الوصف**: تاريخ تسجيل النطاقات
- **البيانات**: معلومات المالك، التواريخ
- **الطريقة**: WHOIS queries + Web scraping
- **المعلومات**: المالك، البريد الإلكتروني، التواريخ

### المرحلة الرابعة: مصادر الصور والملفات

#### 11. Reverse Image Search (reverse-image)
- **الوصف**: البحث العكسي عن الصور
- **البيانات**: مصادر الصورة، الصور المشابهة
- **الطريقة**: Google Images + Yandex + TinEye
- **المعلومات**: مصدر الصورة، الاستخدامات، التعديلات

#### 12. EXIF Data Extractor (exif-extractor)
- **الوصف**: استخراج البيانات الوصفية
- **البيانات**: معلومات الكاميرا، الموقع، التاريخ
- **الطريقة**: ExifTool
- **المعلومات**: GPS، الكاميرا، البرامج المستخدمة

#### 13. Document Metadata (doc-metadata)
- **الوصف**: استخراج بيانات المستندات
- **البيانات**: المؤلف، التاريخ، البرنامج
- **الطريقة**: ExifTool + PyPDF2
- **المعلومات**: معلومات المؤلف، تاريخ الإنشاء

### المرحلة الخامسة: مصادر البيانات المسربة

#### 14. Breach Directory (breach-directory)
- **الوصف**: البحث في التسريبات
- **البيانات**: بيانات مسربة
- **الطريقة**: Web scraping من مواقع التسريبات
- **المعلومات**: كلمات مرور، بيانات شخصية

#### 15. Dehashed Search (dehashed-search)
- **الوصف**: البحث في قواعد البيانات المخترقة
- **البيانات**: بيانات مسربة
- **الطريقة**: Web scraping (free tier)
- **المعلومات**: بريد إلكتروني، كلمات مرور، أسماء

### المرحلة السادسة: مصادر الموقع الجغرافي

#### 16. IP Geolocation (ip-geolocation)
- **الوصف**: تحديد موقع IP
- **البيانات**: الموقع الجغرافي، ISP
- **الطريقة**: ip-api.com (مجاني)
- **المعلومات**: البلد، المدينة، ISP، الإحداثيات

#### 17. WiFi Geolocation (wifi-geolocation)
- **الوصف**: تحديد موقع WiFi
- **البيانات**: مواقع نقاط WiFi
- **الطريقة**: WiGLE (free tier)
- **المعلومات**: إحداثيات نقاط الوصول

## معايير الاختيار

1. **بدون API مدفوع**: جميع المصادر مجانية 100%
2. **معلومات حقيقية**: بيانات فعلية وليست تجريبية
3. **كمية كبيرة**: مصادر غنية بالمعلومات
4. **قانونية**: مصادر عامة ومتاحة للجميع
5. **موثوقية**: مصادر معروفة وموثوقة
