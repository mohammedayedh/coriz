## 🔍 تفاصيل الأدوات العاملة

### 1. IP Lookup 🌐
- **الوصف:** استعلام عن معلومات عنوان IP
- **الخدمة:** ip-api.com (مجانية)
- **المدخلات:** IP address أو hostname
- **المخرجات:** موقع جغرافي، ISP، ASN، منظمة

### 2. Domain Recon 🔍
- **الوصف:** استطلاع معلومات النطاق
- **الخدمة:** hackertarget.com (مجانية)
- **المدخلات:** domain name
- **المخرجات:** WHOIS، DNS records، IP resolution

### 3. Email Scanner 📧
- **الوصف:** فحص صحة البريد الإلكتروني
- **الخدمة:** hackertarget.com + Python
- **المدخلات:** email address
- **المخرجات:** صحة الصياغة، MX records، حل النطاق

### 4. VirusTotal Scan 🛡️
- **الوصف:** فحص الروابط والملفات
- **الخدمة:** VirusTotal API v3
- **المدخلات:** URL أو file hash
- **المخرجات:** نتائج من 75+ محرك فحص
- **ملاحظة:** يعمل بنمط تجريبي بدون API key

### 5. Threat Intelligence ⚠️
- **الوصف:** تغذية استخبارات التهديدات
- **الخدمة:** AlienVault OTX
- **المدخلات:** لا يوجد (تغذية تلقائية)
- **المخرجات:** أحدث التهديدات، APT groups، IOCs
- **ملاحظة:** يعمل بنمط تجريبي بدون API key

### 6. Hash Generator #️⃣
- **الوصف:** مُولّد الهاش المتقدم
- **التقنية:** JavaScript (Web Crypto API)
- **المدخلات:** نص أو ملف
- **المخرجات:** MD5, SHA-1, SHA-256, SHA-512

### 7. Coder/Decoder 🔓
- **الوصف:** تشفير وفك تشفير البيانات
- **التقنية:** JavaScript
- **المدخلات:** نص
- **المخرجات:** Base64, URL, Hex, HTML entities

### 8. Password Generator 🔐
- **الوصف:** مُولّد كلمات المرور الآمنة
- **التقنية:** JavaScript (Crypto.getRandomValues)
- **المدخلات:** طول وخيارات
- **المخرجات:** كلمة مرور آمنة + تقييم القوة

### 9. JWT Inspector 🎫
- **الوصف:** مُفتش توكن JWT
- **التقنية:** JavaScript (jwt-decode)
- **المدخلات:** JWT token
- **المخرجات:** Header, Payload, Signature verification
