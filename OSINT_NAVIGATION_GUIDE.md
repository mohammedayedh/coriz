## 🔍 أدوات الاستخبارات الخادمية

### 1. IP Lookup - استعلام عن IP
```
http://127.0.0.1:8000/osint/intel/ip-lookup/
```
**الاستخدام:**
- أدخل عنوان IP (مثال: 8.8.8.8)
- أو أدخل hostname (مثال: google.com)
- اضغط "تحليل"

**النتائج:**
- الموقع الجغرافي (البلد، المدينة)
- ISP و ASN
- المنظمة
- إحداثيات GPS

---

### 2. Domain Recon - استطلاع النطاق
```
http://127.0.0.1:8000/osint/intel/domain-recon/
```
**الاستخدام:**
- أدخل اسم نطاق (مثال: example.com)
- اضغط "استطلاع"

**النتائج:**
- WHOIS information
- DNS records (A, MX)
- IP resolution
- معلومات المسجل

---

### 3. Email Scanner - فحص البريد الإلكتروني
```
http://127.0.0.1:8000/osint/intel/email-scanner/
```
**الاستخدام:**
- أدخل عنوان بريد إلكتروني
- اضغط "فحص"

**النتائج:**
- صحة الصياغة
- سجلات MX
- حل النطاق
- تقييم الصحة

---

### 4. VirusTotal Scan - فحص الروابط والملفات
```
http://127.0.0.1:8000/osint/intel/virustotal/
```
**الاستخدام:**
- اختر نوع الفحص (URL أو Hash)
- أدخل الهدف
- اضغط "فحص"

**النتائج:**
- إحصائيات من 75+ محرك
- التهديدات المكتشفة
- رابط التقرير الكامل

**ملاحظة:** يعمل بنمط تجريبي بدون API key

---

### 5. Threat Intelligence - استخبارات التهديدات
```
http://127.0.0.1:8000/osint/intel/threat-intel/
```

**ملاحظة:** يعمل بنمط تجريبي بدون API key

## 🔑 تفعيل API Keys (اختياري)

### لتفعيل النتائج الحقيقية (غير التجريبية):

1. **افتح ملف `.env`** في جذر المشروع
2. **أضف المفاتيح:**

```env
# VirusTotal API Key
VIRUSTOTAL_API_KEY=your-virustotal-api-key-here

# AlienVault OTX API Key
OTX_API_KEY=your-otx-api-key-here
```

3. **احصل على المفاتيح من:**
   - VirusTotal: https://www.virustotal.com/gui/join-us
   - AlienVault OTX: https://otx.alienvault.com/

4. **أعد تشغيل الخادم**

## 🚀 نصائح للاستخدام الأمثل

### 1. الأدوات الخادمية
- تحتاج اتصال إنترنت
- قد تستغرق بضع ثوانٍ
- النتائج تُحفظ في قاعدة البيانات

### 2. الأدوات العميلية
- تعمل بدون إنترنت
- فورية (لا انتظار)
- لا تُحفظ في قاعدة البيانات

### 3. إدارة الجلسات
- كل استعلام يُنشئ جلسة جديدة
- يمكنك مراجعة الجلسات القديمة
- يمكنك تصدير النتائج (JSON, CSV)

### 4. القضايا
- أنشئ قضية لكل تحقيق
- اربط الجلسات بالقضايا
- نظّم عملك بشكل أفضل

## ❓ استكشاف الأخطاء

### خطأ 403 (CSRF)
- تأكد من تسجيل الدخول
- امسح الكوكيز وأعد المحاولة
- تأكد من تفعيل JavaScript

### خطأ 500 (Server Error)
- تحقق من اتصال الإنترنت
- تحقق من سجلات الخادم
- تأكد من صحة API keys (إن وُجدت)

### الأداة لا تعمل
- تحقق من المتطلبات
- تحقق من المسار الصحيح
- راجع `OSINT_TOOLS_FIX_REPORT.md`
