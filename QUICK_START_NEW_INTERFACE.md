# 🚀 دليل البدء السريع - الواجهة المبسطة

## ⚡ 3 خطوات فقط للبدء

### الخطوة 1: تشغيل السيرفر
```bash
python manage.py runserver
```

### الخطوة 2: فتح الواجهة
```
http://localhost:8000/osint/search/
```

### الخطوة 3: ابدأ البحث!
```
أدخل أي بيانات → اضغط "ابحث الآن" → شاهد النتائج!
```

---

## 🎯 أمثلة سريعة

### بحث عن بريد إلكتروني
```
الإدخال: test@example.com
الزر: ابحث الآن

```

### بحث عن رقم هاتف
```
الإدخال: +967123456789

```

## 💡 نصائح سريعة

1. **اترك النوع على "تلقائي"** - النظام سيحدد النوع تلقائياً
2. **جرب الأمثلة السريعة** - اضغط على أي مثال للتجربة
3. **انتظر قليلاً** - بعض الأدوات تستغرق دقيقة أو دقيقتين
4. **شاهد التقدم** - كل أداة لها شريط تقدم خاص
5. **افتح الروابط** - اضغط على "فتح" لرؤية التفاصيل

---

## ❓ أسئلة شائعة

### س: كم يستغرق البحث؟
**ج:** من 30 ثانية إلى 3 دقائق حسب نوع البحث

### س: هل يمكنني البحث عن أي شيء؟
**ج:** نعم! بريد، اسم، هاتف، نطاق، IP

### س: هل النتائج دقيقة؟
**ج:** نعم، من مصادر موثوقة ومتعددة

### س: هل يمكنني حفظ النتائج؟
**ج:** نعم، تلقائياً في قسم "الجلسات"

### س: هل الواجهة مجانية؟
**ج:** نعم، مجانية تماماً!

---

## 🎨 ما الجديد؟

### قبل (الواجهة القديمة)
```
1. اختر أداة محددة
2. اذهب لصفحة الأداة
3. املأ النموذج
4. اضغط تشغيل
5. انتظر
6. اذهب لصفحة النتائج
7. شاهد النتائج
```

### الآن (الواجهة الجديدة)
```
1. أدخل البيانات
2. اضغط "ابحث الآن"
3. شاهد النتائج فوراً!
```
