## 🔄 كيفية العمل

### 1. اكتشاف نوع البيانات
```javascript
if (query.includes('@')) → email
else if (query.match(/^\+?\d+$/)) → phone
else if (query.match(/^(\d{1,3}\.){3}\d{1,3}$/)) → ip
else if (query.includes('.')) → domain
else → username
```
