# دليل البحث لأدوات OSINT

## نظرة عامة
تم تحديث جميع أدوات OSINT لتوفير معلومات دقيقة ومخصصة حول نوع البحث المناسب لكل أداة مع أمثلة واقعية.

## التحديثات المنفذة

### 1. تحديث قاعدة البيانات
تم إضافة معلومات مخصصة لكل أداة في حقل `config_schema`:
- `search_type`: نوع البحث المناسب للأداة
- `example`: مثال واقعي على الهدف
- `description_extra`: وصف إضافي لوظيفة الأداة

### 2. تحديث واجهة المستخدم
تم تحسين صفحة تفاصيل الأداة لعرض:
- نوع البحث المحدد لكل أداة
- مثال واضح ومخصص
- وصف إضافي يشرح وظيفة الأداة

## دليل الأدوات المفصل

### 🔍 أدوات البريد الإلكتروني

#### 1. GHunt
- **نوع البحث**: البريد الإلكتروني Gmail
- **مثال**: `example@gmail.com`
- **الوظيفة**: يبحث في حسابات Google ويستخرج معلومات الملف الشخصي
- **ملاحظة**: يعمل فقط مع عناوين Gmail

#### 2. Infoga
- **نوع البحث**: البريد الإلكتروني
- **مثال**: `user@example.com`
- **الوظيفة**: يجمع معلومات عن البريد الإلكتروني من محركات البحث
- **ملاحظة**: يدعم جميع أنواع البريد الإلكتروني

#### 3. HudsonRock Infostealer Search
- **نوع البحث**: البريد الإلكتروني
- **مثال**: `user@example.com`
- **الوظيفة**: يبحث في قواعد بيانات التسريبات وبرامج سرقة المعلومات
- **ملاحظة**: متخصص في اكتشاف التسريبات الأمنية

#### 4. Holehe Account OSINT
- **نوع البحث**: البريد الإلكتروني
- **مثال**: `user@example.com`
- **الوظيفة**: يكتشف الحسابات المرتبطة بالبريد الإلكتروني في مواقع مختلفة
- **ملاحظة**: يفحص أكثر من 120 موقع

#### 5. EmailRep Reputation Analysis
- **نوع البحث**: البريد الإلكتروني
- **مثال**: `user@example.com`
- **الوظيفة**: يحلل سمعة البريد الإلكتروني ويكتشف الأنشطة المشبوهة
- **ملاحظة**: مفيد لتقييم مصداقية البريد

#### 6. LeakPeek Data Breach
- **نوع البحث**: البريد الإلكتروني أو اسم المستخدم
- **مثال**: `user@example.com` أو `username`
- **الوظيفة**: يبحث في قواعد بيانات التسريبات والاختراقات
- **ملاحظة**: يدعم البحث بالبريد أو اسم المستخدم

### 👤 أدوات اسم المستخدم

#### 1. Sherlock
- **نوع البحث**: اسم المستخدم
- **مثال**: `john_doe`
- **الوظيفة**: يبحث عن اسم المستخدم عبر أكثر من 300 موقع تواصل اجتماعي
- **ملاحظة**: الأداة الأكثر شمولاً للبحث عن أسماء المستخدمين

#### 2. Maigret
- **نوع البحث**: اسم المستخدم
- **مثال**: `username123`
- **الوظيفة**: يبحث عن اسم المستخدم في أكثر من 2500 موقع
- **ملاحظة**: أكثر شمولاً من Sherlock

#### 3. CheckNames Domain OSINT
- **نوع البحث**: اسم المستخدم أو النطاق
- **مثال**: `username` أو `example.com`
- **الوظيفة**: يتحقق من توفر اسم المستخدم أو النطاق عبر منصات متعددة
- **ملاحظة**: مفيد لتسجيل الأسماء والنطاقات

### 🌐 أدوات النطاق والـ DNS

#### 1. Harvester
- **نوع البحث**: النطاق
- **مثال**: `example.com`
- **الوظيفة**: يجمع البريد الإلكتروني والنطاقات الفرعية من محركات البحث
- **ملاحظة**: أداة شاملة لجمع المعلومات

#### 2. Subdomain Enumerator
- **نوع البحث**: النطاق
- **مثال**: `example.com`
- **الوظيفة**: يكتشف النطاقات الفرعية للموقع المستهدف
- **ملاحظة**: متخصص في اكتشاف Subdomains

#### 3. MxToolbox DNS & Email Analyzer
- **نوع البحث**: النطاق أو البريد الإلكتروني
- **مثال**: `example.com` أو `user@example.com`
- **الوظيفة**: يحلل سجلات DNS وMX للنطاق
- **ملاحظة**: مفيد لتحليل البنية التحتية للبريد

#### 4. ViewDNS Ultimate Toolkit
- **نوع البحث**: النطاق أو IP
- **مثال**: `example.com` أو `8.8.8.8`
- **الوظيفة**: مجموعة أدوات DNS شاملة للتحليل والاستعلام
- **ملاحظة**: يوفر أكثر من 20 أداة DNS

## نصائح الاستخدام

### 1. اختيار الأداة المناسبة
- حدد نوع الهدف أولاً (بريد، اسم مستخدم، نطاق، إلخ)
- اختر الأداة المتخصصة في هذا النوع
- استخدم الأدوات الشاملة للبحث الواسع

### 2. تنسيق الإدخال
- **البريد الإلكتروني**: استخدم الصيغة الكاملة `user@domain.com`
- **اسم المستخدم**: بدون @ أو رموز خاصة
- **النطاق**: بدون http:// أو www (مثل: `example.com`)
- **IP**: بالصيغة العادية `192.168.1.1`
- **رقم الهاتف**: بالصيغة الدولية `+967779220783`

### 3. الأمثلة الصحيحة

#### ✅ صحيح
- `user@gmail.com` (للبريد الإلكتروني)
- `john_doe` (لاسم المستخدم)
- `example.com` (للنطاق)
- `192.168.1.1` (لعنوان IP)
- `+967779220783` (لرقم الهاتف)

#### ❌ خطأ
- `@user@gmail.com` (رمز @ زائد)
- `http://example.com` (بروتوكول غير مطلوب)
- `192.168.1.1:8080` (منفذ غير مطلوب)
- `0501234567` (رقم هاتف بدون رمز الدولة)
